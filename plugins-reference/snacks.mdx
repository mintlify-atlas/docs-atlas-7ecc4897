Snacks is a modern plugin by Folke that provides a collection of small, useful UI enhancements and utilities. nvim-luffy uses it primarily for smooth scrolling animations.

## Purpose

Snacks adds polish to the nvim-luffy experience:

- Smooth animated scrolling
- Enhanced visual feedback
- Lightweight and modular
- Multiple utilities in one plugin

<Note>
Snacks is part of Folke's ecosystem (same author as Flash, Trouble, and other popular plugins). It provides small enhancements that make Neovim feel more modern.
</Note>

## Configuration

<Accordion title="View Full Configuration">
```lua ~/workspace/source/lua/leyland/plugins/snacks.lua
return {
	"folke/snacks.nvim",
	priority = 1000,
	lazy = false,
	opts = {
		scroll = {
			enabled = true,
			animate = {
				duration = { step = 10, total = 100 },
				easing = "linear",
			},
		},
	},
}
```
</Accordion>

## Enabled Features

### Smooth Scrolling

The primary feature enabled in nvim-luffy:

```lua
scroll = {
  enabled = true,
  animate = {
    duration = { step = 10, total = 100 },
    easing = "linear",
  },
}
```

This provides smooth animated scrolling when using:
- `Ctrl+d` / `Ctrl+u` (half-page scroll)
- `Ctrl+f` / `Ctrl+b` (full-page scroll)
- `gg` / `G` (top/bottom)
- `zz` / `zt` / `zb` (center/top/bottom)

## Key Settings

### Priority Loading

```lua
priority = 1000
lazy = false
```

Snacks loads early and synchronously to ensure smooth scrolling is available immediately.

### Animation Duration

- **step: 10ms** - Duration of each animation frame
- **total: 100ms** - Total scroll animation duration

<Tip>
The 100ms total duration provides smooth scrolling that feels responsive without being too slow.
</Tip>

### Easing Function

```lua
easing = "linear"
```

Linear easing provides consistent scroll speed throughout the animation. Other options include:
- `quadratic` - Slow start, fast end
- `cubic` - More pronounced acceleration

## Performance Impact

Snacks is designed to be lightweight:

- Minimal CPU usage (animations use efficient rendering)
- No lag or stuttering
- Disables automatically if performance drops
- No impact on Neovim startup time

<Tip>
The 100ms animation duration is optimized for modern hardware. On older machines, it still performs well.
</Tip>

## Customization

### Adjust Animation Speed

<Accordion title="Faster/Slower Scrolling">
```lua
-- Edit lua/leyland/plugins/snacks.lua

-- Faster (50ms total)
animate = {
  duration = { step = 5, total = 50 },
  easing = "linear",
}

-- Slower (200ms total)
animate = {
  duration = { step = 10, total = 200 },
  easing = "linear",
}
```
</Accordion>

### Change Easing

<Accordion title="Different Animation Feel">
```lua
-- Quadratic easing (accelerates at end)
easing = "quadratic"

-- Cubic easing (more dramatic acceleration)
easing = "cubic"

-- Linear easing (constant speed)
easing = "linear"
```
</Accordion>

### Disable Smooth Scrolling

<Accordion title="Turn Off Smooth Scroll">
```lua
-- Option 1: Disable in config
opts = {
  scroll = {
    enabled = false,
  },
}

-- Option 2: Runtime disable
:lua require('snacks').scroll.disable()

-- Option 3: Comment out entire plugin
-- Comment the snacks.lua file in plugins/
```
</Accordion>

- Mouse scrolling
- Trackpad gestures
- Relative line numbers
- Line wrapping
- Split windows

## Accessibility

### Motion Sensitivity

<Note>
If animations cause discomfort, you can disable smooth scrolling while keeping other Snacks features.
</Note>

```lua
-- For users sensitive to motion:
opts = {
  scroll = { enabled = false }
}
```

### Alternative: Instant Scroll

Disabling Snacks returns to Vim's default instant scrolling, which some users prefer for:
- Reduced motion
- Lower latency feel
- Less visual distraction

## Troubleshooting

### Scrolling Feels Laggy

If animations stutter:

```lua
-- Reduce animation duration
animate = {
  duration = { step = 5, total = 50 },
}
```

### Scrolling Too Slow

```lua
-- Speed up animations
animate = {
  duration = { step = 5, total = 50 },
}
```

### Want Instant Scrolling Back

```vim
" Disable at runtime:
:lua require('snacks').scroll.disable()

" Or disable in config
```
